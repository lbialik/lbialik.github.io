Bolete Filter

A React Native mobile app of an existing online mushroom triaging tool co-created with Jack Kenney. The Bolete Filter allows users to narrow down Bolete mushrooms (i.e. those with pores instead of gills) based on critical features from 200+ species down to just a few! Offline compatibility allows use anywhere, including in the middle of the woods with no service!
